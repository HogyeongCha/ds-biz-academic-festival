# Step 2: 고객 프로파일링

## 목적

고객의 인구통계, 행동, 구매 데이터를 다각도로 분석하여 다층적 디지털 페르소나를 구축하고, 추천 시스템의 개인화 기반을 마련한다. 단순한 인구통계 세분화를 넘어, RFM(Recency, Frequency, Monetary)과 관여도(Involvement Diversity) 지표를 결합한 행동 기반 클러스터링, 그리고 LLM을 활용한 정성적 페르소나 생성까지 3단계 프로파일링을 수행함으로써, 고객의 '누구인가(Who)'와 '왜 사는가(Why)'를 동시에 해석할 수 있는 통합 프로파일을 구축한다.

## 방법론

> **참조: Paper §4.1 고객 프로파일링**

본 연구에서는 고객의 특성과 구매 행동을 다각도로 해석하기 위해 세 단계의 고객 프로파일링 파이프라인을 설계하였다. 인구통계 기반 초기 필터링부터 시작해, 행동 데이터를 통한 군집화, 마지막으로 LLM 기반 정성 분석을 통해 최종 페르소나를 도출하였다.

### 1단계: 인구통계 기반 1차 필터링 (Paper §4.1.1)

고객 메타데이터(`customer_details.csv`)를 활용하여 Age, Gender, Location, Preferred Size 정보를 기반으로 고객을 넓은 그룹으로 세분화하였다.

- **Age Group 범주화**: `pd.cut()` 함수를 활용해 연속형 나이 데이터를 이산형 범주로 변환하였다.
  - 10대(18–19세), 20대(20–29세), 30대(30–39세), 40대(40–49세), 50대(50–59세), 60대 이상(60세 이상)
- **기타 변수**: Gender, Location, Preferred Size는 원본 값을 그대로 사용하여 필터링 기준으로 활용하였다.

이 과정을 통해 각 고객 ID에 대해 기초적인 인구통계 프로파일(`age_group`, `gender`, `location`, `preferred_size`)이 태깅되었으며, 이는 후속 행동 데이터 분석 및 페르소나 해석의 기반이 된다.

### 2단계: 행동 기반 2차 K-Means 클러스터링 (Paper §4.1.2)

고객의 실제 행동 데이터를 기반으로 구매 관련 성향을 정량화하고 군집화하였다. 전자상거래 거래 데이터(`E-commerce Sales Data 2024.csv`)와 고객 메타데이터를 결합한 후, 대표적인 고객 가치 평가 지표인 RFM과 관여도 지표를 추출하였다.

**추출 지표:**

| 지표 | 설명 |
|------|------|
| Recency | 고객의 마지막 상호작용일과 기준일 간의 일 수 |
| Frequency | 전체 구매(Purchase) 건수 |
| Monetary | 고객당 평균 구매 금액 (Purchase Amount) |
| Involvement Diversity | 조회(View) 또는 찜(Like)한 고유 상품 수 |

**클러스터링 절차:**

1. 각 지표를 **Min-Max Scaling**으로 0~1 범위로 정규화
2. **K-Means** 알고리즘을 적용하여 고객을 군집으로 분류 (K=5~7 범위 탐색)
3. **Elbow Method**를 기반으로 최적 K를 결정 — Inertia의 2차 차분(second derivative)이 최대인 지점을 선택

**클러스터별 행동 그룹 정의:**

클러스터별 중심좌표(centroids)의 특성을 분석하여 다음과 같은 행동 그룹을 정의하였다.

| Cluster | 클러스터 특성 요약 | 명명된 행동 그룹 |
|---------|-------------------|-----------------|
| 0 | R, F, M, Diversity 모두 높음 | **VIP 탐색가** |
| 1 | R, F 높음 / M, Diversity 낮음 | **저관여 충성고객** |
| 2 | R 낮음 / F, M 높음 | **이탈 우려 고가치 고객** |
| 3 | R 높음 / F, M, Diversity 낮음 | **신규 탐색 고객** |

이러한 클러스터는 이후 LLM 기반 페르소나 해석 단계에서 `behavior_cluster_label`로 활용된다.

### 3단계: LLM 기반 3차 페르소나 생성 (Paper §4.1.3)

앞서 도출한 인구통계 정보와 행동 군집 정보를 LLM 기반 언어 추론 모델에 입력하여 고객의 정성적 특성을 해석하고, 최종 페르소나를 생성하였다.

**입력 구성:**
- 인구통계 정보: `age_group`, `gender`
- 행동 클러스터: `behavior_cluster_label`
- 구매 이력: 구매한 상품의 `Product Name`, `Category` (최대 8개)

**프롬프트 구조:**
```
당신은 전문 마케팅 분석가입니다. 아래 고객 데이터를 바탕으로,
이 고객의 핵심 페르소나를 한 문장으로 정의하고,
주요 관심사와 구매 동기를 추론하여 키워드로 제시해 주세요.

- 나이/성별: {age_group} {gender}
- 행동 그룹: {cluster_label}
- 주요 구매 상품: {purchase_items}
```

**출력:**
- 페르소나 한 문장 정의
- 관심사 키워드 (해시태그 형태)
- 핵심 구매 동기

LLM은 **Gemini 2.5 Flash**를 우선 사용하며(논문 원본은 Gemini 2.0 Flash를 사용하였으나 서비스 종료로 인해 가장 근접한 후속 모델인 2.5 Flash로 대체), API가 없는 환경에서는 로컬 fallback 로직으로 대체된다. 생성된 결과는 구조화된 형태로 파싱되어 고객별 최종 페르소나 정보(`persona`, `interest_keywords`, `purchase_motive`)로 저장된다.

**페르소나 생성 예시:**

| User ID | 페르소나 정의 | 관심사 키워드 | 구매 동기 |
|---------|-------------|-------------|----------|
| 201 | 패션에 막 관심을 갖기 시작한 20대 남성 신규 고객 | #데일리룩 #기본템 #패션입문 | 스타일링, 사회적 소속감 |
| 546 | 정기적으로 유아용품을 구매하는 실용적인 30대 육아맘 | #육아필수템 #가성비 #대용량 | 편의성, 안전성, 비용 절약 |
| 3 | 다양한 취미에 아낌없이 투자하는 고관여 50대 남성 | #새로운취미 #고성능 #전문가용 | 자기계발, 여가, 성취감 |

## 결과 요약

### Elbow Method 기반 최적 K 결정

K=5~7 범위에서 K-Means를 반복 실행하고, 각 K에 대한 Inertia(군집 내 분산 합)를 계산하였다. Inertia의 2차 차분이 최대인 지점(elbow point)을 최적 K로 선택하였으며, `elbow_plot.png`에서 빨간 점선으로 표시된 지점이 이에 해당한다.

### 클러스터별 행동 그룹 특성

각 클러스터의 중심좌표(centroid)를 전체 중앙값(median)과 비교하여 행동 그룹을 명명하였다.

- **VIP 탐색가**: Recency, Frequency, Monetary, Diversity 모두 높음 — 최근까지 활발하게 다양한 상품을 탐색하며 높은 금액을 지출하는 핵심 고객
- **저관여 충성고객**: Recency, Frequency 높음 / Monetary, Diversity 낮음 — 자주 방문하지만 소수의 저가 상품만 반복 구매하는 습관적 고객
- **이탈 우려 고가치 고객**: Recency 낮음 / Frequency, Monetary 높음 — 과거에는 활발했으나 최근 활동이 줄어든 고가치 고객으로, 재활성화 전략이 필요
- **신규 탐색 고객**: Recency 높음 / Frequency, Monetary, Diversity 낮음 — 최근 유입된 신규 고객으로, 아직 구매 패턴이 형성되지 않은 상태

### LLM 페르소나 예시

LLM은 인구통계 + 행동 클러스터 + 구매 이력을 종합하여, 고객을 한 문장으로 정의하고 관심사 키워드와 구매 동기를 추론한다. 예를 들어, "패션에 막 관심을 갖기 시작한 20대 남성 신규 고객"이라는 페르소나는 `age_group=20s`, `cluster_label=신규 탐색 고객`, 구매 이력에 Pants/Clothing, Boots/Footwear가 포함된 데이터로부터 생성된 것이다. 이러한 정성적 해석은 마케팅 전략 수립 시 타겟팅, 추천, 콘텐츠 커스터마이징에 직접 활용 가능하다.

## 회고

### 왜 이 방법을 선택했는가

단일 기준(인구통계 또는 행동 데이터)만으로 고객을 세분화하면, 인구통계 기반은 "같은 연령대라도 전혀 다른 구매 패턴을 보이는 고객"을 구분하지 못하고, 행동 기반은 "왜 그런 행동을 하는지"에 대한 해석이 어렵다는 한계가 있었다. 3단계 접근을 선택한 이유는 이 두 가지 한계를 동시에 보완하기 위해서였다. 인구통계로 큰 틀을 잡고, RFM+관여도로 행동 패턴을 정량화하며, LLM으로 정성적 해석을 더하는 구조는 해석 가능성과 예측력을 극대화할 수 있는 설계였다.

### 잘된 점

- **LLM 활용한 정성 분석**: 정량 데이터만으로는 불가능한 '스토리 기반' 고객 해석이 가능해졌다. "정기적으로 유아용품을 구매하는 실용적인 30대 육아맘"이라는 페르소나는 RFM 수치만으로는 도출할 수 없는 마케팅 인사이트를 제공한다.
- **RFM+관여도 결합**: 전통적인 RFM 분석에 Involvement Diversity(조회/찜한 고유 상품 수)를 추가함으로써, 구매 행동뿐 아니라 탐색 행동까지 반영한 다각적 행동 패턴 분석이 가능해졌다. 이를 통해 "많이 사지는 않지만 다양하게 탐색하는 고객"과 "소수 상품만 반복 구매하는 고객"을 구분할 수 있었다.

### 개선할 점

- **LLM 프롬프트 최적화 부족**: 현재 프롬프트는 단일 템플릿으로 모든 고객에 동일하게 적용된다. 클러스터별로 차별화된 프롬프트를 설계하거나, few-shot 예시를 추가하면 페르소나의 일관성과 품질을 더 높일 수 있었을 것이다.
- **클러스터 수 결정의 주관성**: Elbow Method는 시각적 판단에 의존하는 부분이 있어, Silhouette Score나 Gap Statistic 등 추가 지표를 병행했다면 K 결정의 객관성을 더 확보할 수 있었다.

### 배운 점

- **정량+정성 분석 결합의 시너지**: RFM 클러스터링이 "어떤 고객인가"를 수치로 보여준다면, LLM 페르소나는 "왜 그런 고객인가"를 이야기로 풀어준다. 이 두 가지를 결합했을 때 비로소 마케팅 실무에서 바로 활용 가능한 수준의 고객 이해가 가능해진다는 것을 체감하였다.
- **페르소나 마케팅의 실무 적용 가능성**: 생성된 페르소나는 단순한 분석 결과물이 아니라, CRM 캠페인 설계, 타겟 광고 메시지 작성, 상품 큐레이션 전략 수립 등 실무 마케팅 시나리오에 직접 투입할 수 있는 형태였다. 데이터 분석과 마케팅 전략 사이의 간극을 LLM이 효과적으로 메워줄 수 있다는 가능성을 확인하였다.
